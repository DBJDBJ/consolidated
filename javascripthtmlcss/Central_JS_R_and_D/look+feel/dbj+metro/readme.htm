jqMetro

JQUERY PLUGIN FOR METRO UI CONTROLS    ‹ ›

features

- pivot control

Using this plugin you can convert your categorized items into a pivot-style view. This page itself is a good example of how the pivot control will look like in your browser. Pivot control plugin has the following features:
  • Progressive Enhancement: even if javascript is turned off, your content will show reasonably OK to the user.
  • Programmability: Functions that allow you to navigate between different pivot items from javascript.
  • Events: The plugin provides you a set of events which are triggered after different actions in pivot control.
  • Flexible Configurations: You can customize the behaviour of the plugin in regard to generating the pivot control and also the layout of the output HTML.
  • It is possible to create several nested pivot containers.

usage

- pivot control

  1. Download the latest version of jqMetro from download section and the latest version of jQuery from jQuery website.
  2. Add a reference to script and stylesheet files: 
    <head>
    ...
    <script type="text/javascript" src="http://code.jquery.com/jquery-1.5.1.min.js"></script>
    <script type="text/javascript" src="jquery.metro.js"></script>
    <link type="text/css" rel="Stylesheet" href='jquery.metro.css' />
    ...
</head>
  3. Organize your HTML markup as below 
    <div class='metro-pivot'>
    <div class='pivot-item'>
        <h3>your header text</h3> 
        ... your content
    </div>
    ... more pivot items
</div>
  4. Initialize the control by adding the following script to a script block on your page. 
    $(function () {
    $("div.metro-pivot").metroPivot();
});
  5. That's it! You are done!

- theme and accent color

  • To support theme and accent color on your page add the following attribute(s) to the body tag in your HTML: 
    <body accent='orange' theme='light'>
    ... your content
</body>
  • Possible values for theme are: { light, dark }
  • Possible values for accent color are: { orange, red, green, blue, magenta, purple, pink, teal, lime, brown }
  • To use the accent color as the background color or foreground color of an element add the following class(es) to your element: 
    <h1 class='accent-color'>jqMetro</h1>

<div class='accent-bg'>...your content...</div>
  • To use the theme as the background color of an element add the following class to your element: 
    <div class='theme-bg'>...your content...</div>

advance

- pivot control configuration

  1. You can use any of the following configuration parameters when setting up the pivot control to customize different parts and behaviours of your UI. 
    var defaults = {
    animationDuration: 150,
    headerOpacity: 0.5,
    fixedHeaders: false,
    headerSelector: function (item) { return item.children("h3").first(); },
    itemSelector: function (item) { return item.children(".pivot-item"); },
    headerItemTemplate: function () { return $("<span class='header' />"); },
    pivotItemTemplate: function () { return $("<div class='pivotItem' />"); },
    itemsTemplate: function () { return $("<div class='items' />"); },
    headersTemplate: function () { return $("<div class='headers' />"); },
    controlInitialized: undefined,
    selectedItemChanged: undefined
};
